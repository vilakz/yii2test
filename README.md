Вид получается такой :
```
[site_status] => 0
[_eav] => Array (
    [0] => Array (
        [id] => 19

        [value] => Array (
            [0] => 19
        )
    )
    [1] => Array (
        [id] => 2
        [label] => Хитрый аттрибут для теста
        [value] => Array (
            [0] => 2

        )
    )
```
